01시스템 소개

QuoteMaker는 ㈜하나플랫폼의 견적서 자동화 시스템입니다.
행사/렌탈 정보를 입력하면 필요한 기능 모듈이 자동으로 조합되어 견적서가 생성됩니다.

전체 플로우

로그인
→
견적 요청 작성
→
제출
→
개발팀 검토
→
승인
/반려
작동 방식
사업팀이 견적 요청을 제출하면 개발팀에 이메일 알림이 전송됩니다.
개발팀이 검토 후 승인 또는 반려하며, 결과는 견적 현황 페이지에서 확인할 수 있습니다.

02로그인

시스템에 접속하면 로그인 화면이 표시됩니다.

로그인 화면

로그인 방법

  1. 아이디를 입력합니다. (사전에 개발팀에서 발급한 계정)
  2. 비밀번호를 입력합니다.
  3. 로그인 버튼을 클릭합니다.
참고
아이디와 비밀번호를 모두 입력해야 로그인 버튼이 활성화됩니다.
계정이 없는 경우 개발팀에 문의해주세요.

로그인 후 표시되는 메인 화면입니다. 3가지 주요 기능에 바로 접근할 수 있습니다.

로그인 후 홈 화면 (사업팀 기준)
메뉴설명

05견적 요청 - 기능 선택

2 4단계 중 두 번째

행사에 필요한 기능 모듈을 선택합니다. 클릭하여 선택/해제할 수 있습니다.

기능 모듈 선택 화면 (체크 표시 = 선택된 항목)

모듈 선택 방법

  1. 필요한 기능을 클릭하여 선택합니다. (파란 체크 표시로 변경)
  2. 다시 클릭하면 선택 해제됩니다.
  3. 화면 하단에 실시간 가격이 자동 계산됩니다. (소계 + VAT 10% = 합계)
자동 포함 항목
"테스트 및 유지보수"는 모든 견적에 자동 포함됩니다. (초록색 "자동 포함" 배지)
카메라, QR 업로드, 텍스트 입력, 이미지 선택/인쇄 기능 선택 시 "인쇄/출력 및 Printer SDK 연동"이 자동 추가됩니다.
"협의" 표시 항목
AI 화풍변환 등 일부 모듈은 금액이 "협의"로 표시됩니다. 이는 행사 규모에 따라 개발팀과 별도 협의가 필요한 항목입니다.

07견적 요청 - 미리보기 및 제출

4 4단계 중 마지막

입력한 모든 정보를 최종 확인한 후 제출합니다.

견적 미리보기 화면

확인 사항

  1. 기본 정보: 행사명, 일정, 장소, 요청자 정보 확인
  2. 견적 항목: 선택한 모듈과 금액 (소계 + VAT = 합계)
  3. 상세 옵션: 기기, 해상도, 화면 구성, 프린터, 네트워크 확인
제출 전 반드시 확인하세요
제출 후에는 직접 수정이 불가능합니다. 수정이 필요한 경우 "이전" 버튼으로 돌아가서 수정해주세요.

모든 내용이 정확하면 "견적 요청 제출" 버튼을 클릭합니다.

  • "홈으로" 또는 "새 견적 요청" 버튼으로 이동할 수 있습니다.

13자주 묻는 질문 (FAQ)

견적번호는 어디서 확인하나요?
견적 제출 완료 화면에서 바로 확인할 수 있으며, "내 견적 현황" 목록에서도 확인 가능합니다.
제출한 견적을 수정할 수 있나요?
제출 후에는 직접 수정이 불가능합니다. 개발팀에서 검토 중 단가나 항목을 수정할 수 있습니다. 큰 변경이 필요한 경우 개발팀에 연락하시거나 새로운 견적을 요청해주세요.
반려된 견적은 어떻게 하나요?
반려 사유를 확인한 후, 내용을 수정하여 새로운 견적을 다시 요청해주세요. 홈 화면 또는 상단의 "+ 새 견적 요청" 버튼으로 새 견적을 작성할 수 있습니다.
Excel 견적서는 어디서 다운로드하나요?
견적 상세 페이지 우측 상단의 "Excel 다운로드" 버튼을 클릭하면 됩니다. 모든 상태의 견적에서 다운로드 가능합니다.
"협의" 항목은 어떻게 처리되나요?
AI 화풍변환 등 "협의" 표시된 모듈은 행사 규모와 조건에 따라 개발팀과 별도로 금액을 협의합니다. 견적 제출 후 개발팀에서 연락드립니다.
파일 첨부는 어떤 형식이 가능한가요?
이미지(JPG, PNG 등), 음성 파일, 문서(PDF, Word, Excel), 동영상(MP4, WebM)을 첨부할 수 있습니다. 최대 20MB까지 가능합니다.
